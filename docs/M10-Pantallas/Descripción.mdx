## Usa pantallas en tus Proyectos:

Aquí te presentamos una descripción de los diferentes tipos de displays mencionados, resaltando sus características y aplicaciones:

**1. Pantalla ST7789:**

* **Visualización a color:**  La pantalla ST7789 es un display LCD TFT de 2.4" a color con una resolución de 240x320 píxeles. Ofrece una visualización vibrante y detallada de imágenes y gráficos.
* **Interfaz de control:** Usa el protocolo SPI para comunicarse con un microcontrolador, permitiendo una integración sencilla en proyectos. 
* **Retroiluminación:** La pantalla ST7789 puede incorporar una retroiluminación LED blanca, que ofrece una visualización clara en diferentes condiciones de luz.

**2. Pantalla ST7735:**

* **Visualización a color:** La pantalla ST7735 es un display LCD TFT de 1.8" a color con una resolución de 128x160 píxeles. Ofrece una visualización a color de imágenes y gráficos, aunque con menor resolución que la ST7789.
* **Interfaz de control:** También usa el protocolo SPI para comunicarse con un microcontrolador. 
* **Retroiluminación:** La pantalla ST7735 puede incorporar una retroiluminación LED blanca o azul, brindando diferentes opciones de visualización.

**3. Pantalla GC8A01:**

* **Visualización a color:** La pantalla GC8A01 es un display LCD TFT a color de 2.4" con una resolución de 240x320 píxeles. Ofrece una visualización a color de imágenes y gráficos, con una resolución similar a la ST7789.
* **Interfaz de control:** Utiliza el protocolo SPI para comunicarse con un microcontrolador. 
* **Retroiluminación:** La pantalla GC8A01 puede incorporar una retroiluminación LED blanca, ofreciendo una visualización clara en diferentes condiciones de luz.

**4. Pantalla SH1106:**

* **Visualización monocromática:** La pantalla SH1106 es un display OLED monocromático de 0.96" con una resolución de 128x64 píxeles. Ofrece una visualización nítida y de alto contraste, especialmente en condiciones de poca luz.
* **Interfaz de control:** Usa el protocolo I2C para comunicarse con un microcontrolador.
* **Bajo consumo de energía:** La tecnología OLED permite un bajo consumo de energía, ideal para dispositivos portátiles y aplicaciones con batería.

**5. Pantalla SSD1306:**

* **Visualización monocromática:** La pantalla SSD1306 es un display OLED monocromático de 0.96" con una resolución de 128x64 píxeles. Ofrece una visualización nítida y de alto contraste, especialmente en condiciones de poca luz.
* **Interfaz de control:** Usa el protocolo I2C para comunicarse con un microcontrolador.
* **Bajo consumo de energía:** La tecnología OLED permite un bajo consumo de energía, ideal para dispositivos portátiles y aplicaciones con batería.

**Aplicaciones:**

* **Interfaz de usuario:** Mostrar información textual y gráficos simples en dispositivos como relojes inteligentes, medidores, dispositivos de control y otros sistemas pequeños.
* **Prototipado:** Crear prototipos de aplicaciones con interfaces de usuario monocromáticas para dispositivos con bajo consumo de energía.
* **Diseño de interfaces:** Diseñar interfaces de usuario con textos y gráficos simples en dispositivos con limitaciones de espacio.
