# MD02-ESP32


El módulo MD02-ESP32 te introduce en el mundo del Internet de las Cosas (IoT) con dos placas ESP32 de gran versatilidad: la ESP32-H2-DevKitM-1 y la ESP32-C6-DEV-KIT-N8.  Estas placas te permiten crear dispositivos inteligentes con conectividad WiFi, Bluetooth y capacidades de procesamiento avanzadas.

## **ESP32-H2-DevKitM-1:  Rendimiento y Conectividad**

* **Procesador Xtensa LX6 Dual-Core:**  Ofrece un alto rendimiento para aplicaciones de IoT.
* **Memoria Flash:**  Amplia capacidad para almacenar programas y datos.
* **Conectividad WiFi y Bluetooth:**  Permite la comunicación inalámbrica con otros dispositivos.
* **Interfaces:**  Soporta UART, I2C, SPI, SD Card, entre otras.
* **GPIO:**  Un gran número de pines de entrada/salida para controlar componentes y sensores.
* **Pantalla LCD:**  Incluye una pantalla LCD integrada para la visualización de información.

## **ESP32-C6-DEV-KIT-N8:  Optimizado para IoT**

* **Procesador Xtensa LX6 Dual-Core:**  Rendimiento optimizado para tareas de IoT.
* **Conectividad WiFi y Bluetooth:**  Conexión inalámbrica para control remoto.
* **Interfaz de Bajo Consumo:**  Ideal para dispositivos alimentados por baterías.
* **Sensores Integrados:**  Incluye sensores de temperatura y proximidad.
* **GPIO:**  Pines de entrada/salida para una amplia gama de aplicaciones.

## **Aplicaciones Prácticas:**

* **Domótica:**  Automatizar el control de luces, electrodomésticos y sistemas de seguridad.
* **Monitoreo Ambiental:**  Capturar datos de sensores de temperatura, humedad, calidad del aire, etc.
* **Control Remoto:**  Controlar dispositivos a distancia a través de aplicaciones móviles.
* **Proyectos Creativos:**  Desarrollar robots, sistemas de alerta temprana, dispositivos de telemetría y mucho más.
* **Aplicaciones Industriales:**  Monitoreo de procesos, control de maquinaria y optimización de sistemas.
