# M01-Arduino

El módulo MD01-Arduino es la base para iniciarse en el emocionante mundo de la programación de microcontroladores. Contiene dos placas Arduino de gran versatilidad: la Arduino MKR1010 y la Arduino Mega, que te permiten experimentar con diferentes niveles de complejidad y realizar proyectos increíbles.

## **Arduino MKR1010: Potencia y Conectividad**

* **Conectividad WiFi:**  Integrada para conectar tus proyectos a internet.
* **Procesador ARM Cortex-M0+:**  Ofrece un alto rendimiento para aplicaciones complejas.
* **Memoria Flash:**  Amplia capacidad para almacenar programas y datos.
* **Comunicaciones:**  Soporta UART, I2C, SPI y Ethernet.

## **Arduino Mega: Versatilidad y Expansión**

* **Microcontrolador ATmega2560:**  Potente para proyectos que requieren un gran número de pines.
* **Número de Pines:**  54 pines digitales (de los cuales 14 pueden utilizarse como salidas PWM), 16 pines analógicos, 4 UARTs y un reloj en tiempo real.
* **Memoria Flash:**  256 KB para almacenamiento de programas.
* **Memoria SRAM:**  8 KB de memoria RAM para variables y almacenamiento temporal.
* **Memoria EEPROM:**  4 KB de memoria no volátil para almacenamiento de datos.

## **Aplicaciones Prácticas:**

* **Control de Motores:**  Manejar motores DC, servos y motores de pasos.
* **Sensores:**  Leer datos de sensores de temperatura, humedad, luz, presión, etc.
* **Interfaces:**  Interactuar con pantallas LCD, pantallas OLED y TFT.
* **Comunicaciones:**  Comunicarse con otros dispositivos Arduino, ordenadores y teléfonos inteligentes.
* **Redes Inalámbricas:**  Crear proyectos con conectividad WiFi.
* **Proyectos Creativos:**  Desarrollar robots, juegos interactivos, dispositivos de domótica y mucho más.
